Detailed Description

A class for reading values by section and key from a standard ".ini" initialization file.

Section and key names are not case-sensitive. Values are loaded into a hash table for fast access. Use GetAllValues to read multiple values that share the same section and key. Sections in the initialization file must have the following form:

; comment line
[section]
key=value

Examples: var iniFile = new IniFile("MyFile.ini"); string font = iniFile.GetValue("Text Style", "Font", "Arial"); int size = iniFile.GetInteger("Text Style", "Size", 12); bool bold = iniFile.GetBoolean("Text Style", "Bold", false);
