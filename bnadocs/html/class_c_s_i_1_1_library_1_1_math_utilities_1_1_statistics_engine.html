Detailed Description

Iterative Curve-fit for Y= slope*X+ Y0 Calculates slope, Y-intercept, and offset (of the form Y = slope*(X-X0) + Y0) Also, calculates Standard Deviation for X, Y, and r^2 (test for fit)

Usage: 1) Use InitLR(..) to reset 2) UpdateLR( x, y) to input (update) new values. 3) Use ReadR2() to check "goodness of fit" 4) Read required curve-fit parameters

Linear Curve-fit: y(i) = slope*x(i) + Y_intercept Note: If Xoffset is set, y(i) = slope( x(i)+Xoffset ) + Y_intercept

Xoffset can be used to offset X_value, if required.
